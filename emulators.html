What is Emulation?

"In computing, an emulator is hardware or software that enables one computer system to behave like another computer system. An emulator typically enables the host system to run software or use peripheral devices designed for the guest system" - Wiki
In other word, an emulator is a program that can run software from a different computer. Emulation uses the actual game files. Because of this, emulation is NOT simulation.
