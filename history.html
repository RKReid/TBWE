TBWE2

This was when TBWE started to become a good website. It was not an online website yet. It was made to be stored locally. It was created in IST 4 Day. The original is mostly HTML, but the second one (HTML 2a challenge) started using CSS and images. It was the start of all modern TBWEs, as they still use code from it. Later, I discovered how to unblock flash games from y8.com, which lead to the creation of TBWE3, which was meant to share the unblocked games.
