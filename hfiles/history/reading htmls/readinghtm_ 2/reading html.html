How to Write a Simple Website

Writing a website is easier than you think it is. In this HTML, I will teach you how to code your very own website!





Text Editor

Before you begin to code an HTML (a website) you need a text editor. I reccomend using Notepad++ or Caret. You can also use the program Notepad that come with Windows OS, but compared to the other two text editors I listed, Notepad is horrible.




What are Tags

Each line of code that is surrounded by <> is called a tag. Most code in an HTML is written inside of tags. Tag examples < p > < img > < h6 >. Please note: tags are normaly written WITHOUT spaces between the greater/less than signs.

Text

In HTMLs there are many ways of displaying text. Some of the most common methods are to use the tags < h1 > and < p >. < H1 > would display a heading. < P > is used for making paragraphs.
